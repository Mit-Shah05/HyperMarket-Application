# HyperMarket-Application
Web-based hypermarket application with product management, cart functionality, and order handling using PHP and MySQL.

# Overview
This project is a web-based Hypermarket Application developed to simulate an online shopping system. It allows users to browse products across multiple categories, manage a shopping cart, and place orders. The application focuses on backend logic, session management, and database integration.

# Features
User session management

Multiple product categories

Add-to-cart and cart management

Order placement and purchase history

Database-driven product and order storage

# Tech Stack
Frontend: HTML, CSS

Backend: PHP

Database: MySQL

Sessions: PHP Sessions

# Database Structure
Product (ProductID, ProductName, Category, Brand, Price, StockQuantity)

Orders (OrderID, UserName, PaymentMethod, OrderDate)

OrderDetails (OrderID, ProductID, Quantity, Price)

# How It Works
User selects products from different categories

Selected items are stored using session-based cart handling

Orders are placed and stored in the database

Purchase history is maintained for each user

# Learning Outcomes
Hands-on experience with PHP backend development

Understanding of session handling and database design

Practical exposure to full-stack web application flow
